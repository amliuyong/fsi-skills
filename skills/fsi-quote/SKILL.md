# FSI 实时行情

获取 A 股实时行情数据（腾讯财经接口），包括价格、涨跌幅、成交量、换手率、PE/PB、买卖盘等。

## 前置条件

- 已通过 `install.sh` 安装 FSI，`fsi` 命令可用（脚本会自动从 bundled fsi-pkg 安装）
- 需要网络连接（实时数据）
- 无需预先拉取数据

## 用法

```bash
# 查询单只股票
fsi quote 000001

# 查询多只股票
fsi quote 000001 600519 300750

# ETF 行情
fsi quote 510300 159919
```

## 意图识别

| 用户说的 | 示例 |
|----------|------|
| "现在多少钱" / "实时价格" | `fsi quote 000001` |
| "看看报价" / "当前行情" | `fsi quote 600519` |
| "茅台涨了没" / "今天涨跌" | `fsi quote 600519` |
| 同时问多只 | `fsi quote 000001 600519 300750` |

## 输出

JSON 格式，包含：
- `price` — 当前价格
- `pct_change` — 涨跌幅 %
- `change_amt` — 涨跌额
- `open` / `high` / `low` / `prev_close` — 开高低昨收
- `volume` / `amount` / `turnover` — 成交量/成交额/换手率
- `pe` / `pb` / `market_cap` — 市盈率/市净率/总市值
- `bid1` / `ask1` — 买一/卖一
